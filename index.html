Tugas pasca UTS

Bacalah penjelasan pembahasan UTS, unduh dan cobalah ubah file html dan css yang menjadi studi kasus jawaban UTS tersebut.

Masing-masing mahasiswa wajib memberikan minimal dua pertanyaan yang belum pernah ditanyakan sebelumnya mengenai kedua file tersebut melaui email ke dhimas at lepisi.ac.id dengan subjek WDA20TUG5. Jika pertanyaan yang diajukan ternyata sudah pernah diajukan, silakan kirimkan pertanyaan lain hingga kuota pertanyaan mahasiswa terpenuhi. Pertanyaan yang dikirim serta jawaban saya akan saya update pada halaman ini per hari.

Pertanyaan yang diajukan adalah pertanyaan yang memang sulit untuk Anda mengerti. Jika penjelasan yang didapat dari google sudah cukup jelas, maka pertanyaan tersebut tidak perlu diajukan melalui email. Batas maksimal mengirim pertanyaan adalah sebelum UAS.

Mahasiswa diwajibkan membaca buku HTML & CSS Design and Build Websites oleh Jon Duckett yang telah diberikan pada awal pertemuan, buku tersebut juga dapat digunakan sebagai bahan bertanya.

Studi Kasus

Studi kasus adalah sebuah halaman web tunggal untuk website kampus yang terdiri dari html, css dan javascript. Silakan dipelajari melalui komen yang ada dan mencoba mengubah file-file tersebut.

Studi kasus dapat dilihat secara live pada halaman berikut ini. Sedangkan source code dapat dilihat pada halaman:

  • file html
  • file css
  • file javascript

UAS Praktek

Carilah ide tentang sebuah usaha berbasis online berdasarkan pilihan jenis usaha di bawha ini kemudian buatlah sebuah desain website untuk usaha tersebut. Kamu boleh menggunakan gambar dari internet, namun isi website yang berupa teks harus merupakan hasil tulisan sendiri. Cari nama dan moto yang unik untuk usaha tersebut.

  • Koperasi
  • Bengkel sepeda
  • 3D printing
  • Asisten rumah tangga part time
  • Pelabuhan
  • Penitipan anak
  • Krematorium
  • Peternakan

Adapun ketentuan penilaian adalah sebagai berikut:

  1. Website dibuat menggunakan html dan css (nilai tambah jika menggunakan javascript). 20 poin
  2. Website bisa berupa single page (panjang) seperti studi kasus atau multiple page (nilai tambah jika desain telah responsive). 20 poin
  3. Website wajib memiliki menu navigasi. 10 poin
  4. Konten berupa teks bukan hasil menyalin atau teks template seperti lorem ipsum. 20 poin
  5. Website wajib memiliki gambar. 10 poin
  6. Desain website yang dibuat berguna dan sedap dipandang. 20 poin

Pengumpulan

Jawaban UAS dikumpulkan dengan cara:

  1. File html, css dan js dimasukkan ke dalam folder bernama sesuai NIM masing-masing (contoh: 20198040020).
  2. Kompres folder tersebut dengan format zip (contoh: 20198040020.zip).
  3. Kirim melalui email ke dhimas@lepisi.ac.id dengan subjek DWA20UAS, sertakan juga nama kamu di dalam email supaya tidak salah.

Batas akhir pengumpulan jawaban UAS sesuai dengan jadwal UAS dari akademik.
